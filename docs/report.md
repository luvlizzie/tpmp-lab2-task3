### Описание структуры TOVAR
Структура содержит поля:
- `name` - название товара (строка)
- `quantity` - количество единиц (целое число)
- `price` - стоимость (вещественное число)
- `receipt_date` - дата поступления (структура Date с полями year, month, day)

### Реализованные функции
1. `add_tovar` - добавление нового товара
2. `calculate_average_price` - вычисление средней стоимости
3. `sort_by_price` - сортировка по возрастанию цены
4. `print_old_goods` - вывод товаров, поступивших более 10 месяцев назад
5. `print_all_goods` - вывод всех товаров
